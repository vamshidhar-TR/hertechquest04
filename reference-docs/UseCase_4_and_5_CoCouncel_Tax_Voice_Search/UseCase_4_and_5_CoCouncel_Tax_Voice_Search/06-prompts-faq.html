FAQ / common pitfalls

SymptomWhat's happening & fix
pip install fails with 401TR's internal index token is expired. Append --index-url https://pypi.org/simple to install from public PyPI.
"Microphone error" / no audioCheck Windows mic permissions (Settings → Privacy → Microphone) and that a mic is the default input device.
First run hangs for a whileNot frozen — the Whisper model (~75 MB for base) is downloading. It's cached after the first run.
"(no speech detected)"Speak a little longer and closer to the mic. Very short clips get dropped. Try a bigger model: VK_WHISPER_MODEL=small.
Transcript is slightly wrongExpected — edit the text box before sending. For accuracy over speed, use small or medium.
MCP: "Could not connect"You skipped the one-time browser login. Run npx -y mcp-remote <url> --transport http-only once and complete sign-in. Ensure node --version works.
Live answer takes 30–60sNormal for real research tools. Show a spinner/timer; don't block the UI thread on the call.
Citations don't openMake sure your browser is signed in to CoCounsel; confirm the link wasn't rewritten by the client.
UI freezes while transcribingYou're running inference on the UI thread. Move capture, transcription, and the backend call to background threads.
