⏯ Run Scripts in the Background

Keep a voice listener running while you work in another window during a demo.

Windows — PowerShell (recommended here)

Background job

Start-Job -ScriptBlock { python listener.py }
Get-Job                    # list running jobs
Receive-Job -Id 1          # view output from job 1
Stop-Job -Id 1; Remove-Job -Id 1   # stop and clean up

Detached hidden window

$p = Start-Process python -ArgumentList 'listener.py' -WindowStyle Hidden -PassThru
$p.Id                      # note the process ID
Stop-Process -Id $p.Id     # stop it later

Command Prompt alternative: start /b python listener.py — stop via Task Manager or taskkill /IM python.exe /F.

macOS / Linux (or Windows WSL)

nohup — simplest

nohup python listener.py > listener.log 2>&1 &
echo $! > listener.pid     # save the PID
kill $(cat listener.pid)   # stop it later

tmux — best for demos

tmux new -s voice          # start a named session; run your script inside it
# Ctrl+B then D            detach (script keeps running)
tmux attach -t voice       # re-attach later
tmux ls                    # list sessions
Demo tip. On Windows, a PowerShell background job lets you present from a clean window and check logs with Get-Job / Receive-Job. On macOS/Linux, start the listener in tmux, detach, and re-attach to check logs.
